# FBLA-project
## FBLA competition game overview
This Project was made for a middle school FBLA competition on Scratch. This game is about a robot who comes from another plantet and lands on earth to find 3 time machines. In each of those time machines is a different time period which the user can go in and interact with important figures from the time period. There are items scattered around the timeperiod for the user to find and choose to sell in the shop or negotiate prices to sell to the NPC's in the time period. When the user feels they have enough money and reputation they can return back using the UFO. If the user has enough, they have won the game, if not they must restart.

## How the Code works
Much of this code was written event based
Using the event code outline, the program will function when a event is activated
the "say" code displays the sprite's scripts
Some sprites are placed without a set location, but they will go automatically to a certain zone.
When a sprite is clicked or touched by the character, the code will run to display and activate functions

## How to play the game
The game can be played and restarted using the basic green flag icon in the top left corner of the screen.

## Find a Bug?
This game was made in under 10 days, so bugs are most definitely possible. 
If bugs are found submit an improvement to the project.
Commenting on the project about the errors can also help fix the bugs
